## Plano de Execução
1. Criar as subpastas em `frontend/src/components/`.
2. Mover cada um dos arquivos usando `git mv`.
3. Ajustar todos os imports afetados nos próprios componentes movidos e em `frontend/src/App.tsx`.
4. Validar o build com `npm run build` na pasta `frontend/`.
